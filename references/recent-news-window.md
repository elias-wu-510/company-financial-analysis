# Recent News Window

Use this reference for company-analysis tasks when recent developments may materially affect interpretation.

## Default rule

For company-analysis work, proactively search the company’s **most recent three months** of:
- official announcements
- press releases
- investor updates
- relevant public news coverage
- major project or operating updates

Do this even if the user does not explicitly ask for news.

## Goal

Make the report more current without confusing recent narrative with already-reported financial results.

## Required checks

- [ ] Search a recent-news window of roughly the last 3 months.
- [ ] Prefer official company and exchange sources first.
- [ ] Use public news mainly for supplementary context.
- [ ] Check event date, publication date, and relevance to the financial period.
- [ ] Distinguish clearly between:
  - already reflected in reported results
  - likely relevant but not yet visible in reported numbers
  - future pipeline only

## Delivery rule

Recent-news retrieval is mandatory by default for company analysis. Inclusion in the report still depends on evidence strength and period fit.
