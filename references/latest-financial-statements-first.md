# Latest Financial Statements First

Use this reference when the latest full-year annual report has not yet been published, but newer official financial statements are already available.

## Default rule

When the latest full-year annual report has not yet been released, default to using the **latest official disclosed financial statements** as the primary basis for current analysis.

Examples include:
- interim report
- results announcement
- press release with financial tables
- official investor presentation with clearly reported financial figures

Use the prior full-year annual report as:
- baseline
- structural background
- historical comparison anchor

Do **not** let the prior full-year report substitute for the latest financial period when newer official financial statements already exist.

## Analytical implication

For a current-year analysis:
- latest disclosed financial statements = main current-period base
- prior annual report = background and benchmark
- news / announcements / external context = explanation layer

## Delivery rule

State clearly when the current analysis is based on the latest interim or announcement-level financial statements rather than a full-year annual report.
